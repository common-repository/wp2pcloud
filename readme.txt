Backup your WordPress website with all files, media and DB backed up to pCloud. Make it in minutes with this cool and user-friendly plugin.

== Description ==
WordPress Backup to pCloud has been created to backup your blog and its data, regularly. Just choose a day, time and how often you wish your backup to be saved in your pCloud!

== Installation ==
Once installed, the authorization is one step process -
When you first access the plugin’s options page, it will ask you to authorize the plugin with pCloud. Type your email and password and press the button.

= Minimum Requirements =

1. PHP 5.2.16 or higher with [cURL support](http://www.php.net/manual/en/curl.installation.php)
2. [A pCloud account](https://my.pcloud.com/#page=register&ref=1235)
